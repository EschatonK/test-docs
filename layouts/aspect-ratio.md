**- Property: aspect-ratio** abbreviation: **ar**<br>
Utilities for controlling the aspect ratio of an element.

```css
arA {
  aspect-ratio: auto;
}
arS {
  aspect-ratio: 1 / 1;
}
arV {
  aspect-ratio: 16 / 9;
}
```

**Example:**

```html
<img class="arV" src="./layouts/img/nha-trang-beaches-1.webp" />
```
